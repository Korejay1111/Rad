# 🚗 Premium Car Dealer Website

A stunning, fully-responsive premium car dealer online showroom website built with pure HTML, CSS, and JavaScript. This luxurious website features a modern design inspired by high-end automotive brands like Mercedes-Benz and Tesla.

## ✨ Features

### 🏠 Homepage
- **Hero Section**: Large hero section with high-quality car background image and smooth animations
- **Featured Cars**: Showcase of premium vehicles with hover effects and quick view cards
- **Why Choose Us**: Trust indicators highlighting dealership strengths
- **Statistics Counter**: Animated counters displaying key metrics (500+ cars sold, 15+ years experience)
- **Testimonials**: Customer reviews carousel with ratings
- **FAQ Section**: Accordion-style frequently asked questions
- **Floating Buttons**: WhatsApp chat, scroll to top/bottom navigation
- **Dark/Light Mode**: Full theme toggle functionality

### 🚗 Available Cars Page
- **Advanced Filtering System**:
  - Search by car name/model
  - Filter by price range
  - Filter by brand (BMW, Mercedes, Porsche, etc.)
  - Filter by year
  - Filter by fuel type (Petrol, Diesel, Electric, Hybrid)
- **Responsive Grid Layout**: Beautiful car cards with smooth hover animations
- **Quick Actions**: View details and WhatsApp inquiry buttons
- **Favorites Feature**: Save cars to favorites list
- **Results Counter**: Dynamic count showing filtered results

### 📋 Car Details Page
- **Image Gallery**: Interactive gallery with thumbnail navigation and zoom functionality
- **Specifications Grid**: Comprehensive vehicle specifications
- **Finance Calculator**: Monthly payment estimator with adjustable parameters
  - Vehicle price
  - Down payment
  - Interest rate
  - Loan term
- **Features Breakdown**: Detailed list of performance, interior, and safety features
- **Quick Enquiry Form**: Contact form for immediate inquiries
- **WhatsApp Integration**: Pre-filled WhatsApp messages with car details

### 🎨 Categories Page
- **Visual Category Cards**: Stunning category cards with hover effects
- **Categories Include**:
  - Luxury Cars
  - Sports Cars
  - SUVs
  - Sedans
  - Electric Vehicles
  - Budget-Friendly Options
- **Featured Category Section**: Highlighted premium category
- **Vehicle Counts**: Display number of vehicles per category

### 🏢 About Us Page
- **Company Story**: Engaging narrative about dealership history
- **Mission & Vision**: Clear statement of values and goals
- **Core Values**: Integrity, Customer First, Excellence, Teamwork
- **Team Section**: Professional team member profiles with social links
- **Certifications**: Dealer certifications and achievements
- **Statistics Showcase**: Key business metrics

### 📞 Contact Us Page
- **Contact Information**: Address, phone, email, WhatsApp
- **Contact Form**: Comprehensive form with subject selection
- **Google Maps Integration**: Embedded dealership location map
- **Business Hours**: Clear operating schedule
- **Quick Actions**: Test drive scheduling and WhatsApp shortcuts
- **Social Media Links**: Connect on all major platforms

## 🚀 Getting Started

### Installation

1. Clone or download this repository
2. Open `index.html` in a web browser
3. No build process or dependencies required!

### File Structure

```
premium-car-dealer-website/
│
├── index.html              # Homepage
├── cars.html               # Available cars with filters
├── car-details.html        # Individual car details
├── categories.html         # Vehicle categories
├── about.html              # About us page
├── contact.html            # Contact page
├── css/
│   └── styles.css          # Global styles and CSS variables
├── images/                 # (Unsplash URLs used)
└── README.md              # This file
```

### Customization

#### Colors
Edit CSS variables in `css/styles.css`:

```css
:root {
  --primary-gold: #D4AF37;
  --primary-navy: #0A1628;
  --primary-black: #1A1A1A;
  /* ... more variables */
}
```

#### Images
Replace Unsplash URLs with your own car images:
- Hero backgrounds
- Car cards
- Category backgrounds
- Team photos

#### Contact Information
Update contact details in:
- Navigation bar
- Footer
- Contact page
- WhatsApp links
